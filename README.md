## Open Source ESP8266 Based Laser Jammer
* 12V Circuitry for Automobile Install
* 940nm Brute Force Mode
* Wireless Connectivity & Android App
* Legal in SOME STATES!
* Laser Gun Emulation Mode
* MIRT Mode (really bad idea)
